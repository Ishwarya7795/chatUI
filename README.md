# Social Media Chat UI (HTML + CSS)

A simple and modern **social media-style chat user interface** built using **only HTML and CSS**. This project does **not use any JavaScript**, making it lightweight and easy to use.

## Features

- Sidebar with a list of contacts.
- Chat area with sent and received message bubbles.
- Gradient colors and hover effects for a modern look.
- Scrollable chat area and contact list.
- Default avatar fallback for empty or missing profiles.
- Fully responsive and clean layout.

## How to Use

1. Clone this repository or download the HTML file.
2. Open `index.html` in your browser.
3. You will see the chat UI with cartoon avatars, gradients, and styled chat bubbles.
